Drop your gallery images here.

Any image file in this folder (.png, .jpg, .jpeg, .webp, .avif, .gif, .svg)
automatically appears in the "Gallery" carousel on the home page, sorted by
filename. To control the order, prefix names like 01-..., 02-..., etc.

Add files = they show up. Remove files = they disappear. No code changes needed.
If this folder is empty, the Gallery section is hidden entirely.
